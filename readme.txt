Adds an ajaxy search box to the post list page

== Description ==
Adds a "quick search" box to the Posts list page in the WordPress dashboard which uses jQuery-UI autocomplete and ajax call(s) to quick-search thru post titles to go to the edit page. Also works with custom post types.


== Installation ==

= Manual Installation =

1. Upload the entire `/wordpress-post-quick-search` directory to the `/wp-content/plugins/` directory.
2. Activate Admin Post Quick Search through the 'Plugins' menu in WordPress.
